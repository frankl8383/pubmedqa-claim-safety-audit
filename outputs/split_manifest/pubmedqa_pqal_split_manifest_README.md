# PubMedQA PQA-L split manifest

This folder records the fixed split metadata used by the audit package. The public manifest includes split, PMID, label, year, and SHA-256 hashes of the question and context fields rather than redistributing full abstract/context text. The split was generated by `prepare_biomed.py` with `SPLIT_SEED = 20260503`, `TRAIN_FRAC = 0.70`, and `VAL_FRAC = 0.15`.
