# Supervisor-Informed Case Audit

This directory contains submission-facing versions of the 60-case illustrative
case audit. The audit is supervisor-informed and standardized after
review. It is not independent expert consensus, blinded PubMedQA relabeling, or
evidence that the agent improved biomedical reasoning.

Files:

- `supervisor_case_audit_public_supplement_60.tsv`: recommended public
  supplement; excludes full abstract/context text.
- `supervisor_case_audit_standardized_60.tsv`: analysis-facing table with
  PubMedQA questions but no full abstract/context text.
- `supervisor_case_audit_taxonomy_summary.tsv`: case counts by taxonomy class.
- `supervisor_case_audit_main_text_candidates.tsv`: candidate illustrative
  examples for manuscript text.
- `supervisor_case_audit_provenance_sanitized.md`: submission-safe provenance.
